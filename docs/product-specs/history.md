# Product Spec — 분석 이력

**Route:** `/history`  
**대상 사용자:** 로그인한 영업사원  
**진입 경로:** 사이드바 최근 항목 전체 보기 / 직접 URL 접근

---

## 목적

영업사원이 과거에 분석한 기업 목록을 한눈에 보고, 원하는 분석으로 빠르게 재접근하는 화면.

---

## 화면 구성

```
┌────────────────────────────────────┐
│  페이지 타이틀 + 검색창              │
├────────────────────────────────────┤
│  필터 탭 (전체 / 즐겨찾기)           │
├────────────────────────────────────┤
│  분석 카드 목록                      │
│  (최신순 정렬, 무한 스크롤)           │
└────────────────────────────────────┘
```

---

## 섹션 상세

### 헤더

- 페이지 타이틀: "분석 이력"
- 검색창: 기업명 실시간 필터링 (클라이언트사이드)

---

### 필터 탭

- **전체** — 모든 분석 이력
- **즐겨찾기** — 별표 표시한 분석만

---

### 분석 카드

각 카드 구성:

| 항목 | 설명 |
|---|---|
| 기업 이니셜 아이콘 | 기업명 첫 글자, 색상 랜덤 고정 |
| 기업명 | 굵게 |
| 업종 | 작은 텍스트 |
| 결산 기준일 | 예: 2024-12-31 결산 |
| PDF 포맷 | CRETOP / NICE / 일반 배지 |
| 분석 완료 시각 | 상대 시간 (예: 3일 전) |
| 즐겨찾기 별 아이콘 | 클릭 시 즐겨찾기 토글 |
| 상태 배지 | 완료 / 분석 중 / 오류 |

카드 클릭 시 → 해당 `/dashboard/[id]`로 이동.

---

### 빈 상태

분석 이력이 0건일 때:
- "아직 분석한 기업이 없어요" 메시지
- "첫 분석 시작하기" 버튼 → 업로드 모달 열기

즐겨찾기 탭에서 즐겨찾기가 0건일 때:
- "즐겨찾기한 분석이 없어요" 메시지
- "브리핑 화면에서 ☆를 눌러 즐겨찾기를 추가해보세요" 안내

---

### 정렬 및 페이지네이션

- 기본 정렬: 최신 분석 순
- 무한 스크롤 (페이지 버튼 없음) — 20건씩 로드
- 검색 필터는 클라이언트사이드에서 처리 (이미 로드된 목록 내 필터링)

---

## 오류 처리

| 상황 | 처리 |
|---|---|
| 데이터 로드 실패 | "이력을 불러오지 못했어요. 새로고침 해주세요." |
| 비인증 접근 | `/auth/login`으로 리디렉트 |
