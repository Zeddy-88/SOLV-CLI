# Product Spec — 신규 사용자 온보딩

**대상:** 처음 가입한 영업사원  
**목표:** 가입 후 첫 분석 완료까지 이탈 없이 도달시키기

---

## 온보딩 플로우

```
회원가입 완료
    │
    ▼
웰컴 화면 (단 1회 노출)
    │
    ▼
메인 화면 (/) — 온보딩 가이드 오버레이
    │
    │  PDF 업로드
    ▼
첫 분석 완료 → 온보딩 종료
```

---

## 1단계 — 웰컴 화면

회원가입 직후 단 1회만 표시. 이후 재방문 시 노출하지 않는다.

**구성:**
- 앱 로고 + 서비스명
- 환영 메시지: "영업 준비, 이제 1분이면 충분해요"
- 서비스 핵심 가치 3줄 요약:
  - PDF 업로드 → 재무 분석 자동화
  - 4인 전문가 다각도 분석
  - 치알디니 7원칙 기반 영업 멘트 생성
- "시작하기" 버튼 → 메인 화면으로 이동

**조건:** `localStorage` 또는 DB에 `onboarding_completed: false` 상태일 때만 표시.

---

## 2단계 — 메인 화면 첫 방문 가이드

분석 이력이 0건인 사용자에게만 표시하는 업로드 유도 UI.

**드롭존 위 안내 배너:**
- 텍스트: "첫 번째 기업을 분석해보세요"
- 서브: "CRETOP, NICE BizLine, 일반 재무제표 PDF를 업로드하면 AI가 영업 브리핑을 만들어드려요."
- 샘플 분석 보기 링크 (선택): 샘플 데이터로 결과 화면 미리보기 제공

**첫 분석 완료 시:**
- 온보딩 상태 `completed`로 업데이트
- 이후 방문부터는 일반 메인 화면으로 진입

---

## 이탈 방지 원칙

- 온보딩 도중 강제 튜토리얼 없음 — 사용자가 원하면 언제든 건너뛸 수 있다
- 가입 후 바로 핵심 기능(업로드)에 접근할 수 있도록 화면 수를 최소화
- 온보딩 완료 여부는 서버(Supabase DB)에 저장 — 기기를 바꿔도 재노출되지 않음
