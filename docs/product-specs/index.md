# Product Specs — Index

법인 컨설팅 영업 지원 AI 툴의 전체 제품 스펙 목록과 페이지 구조.

---

## 제품 목표

영업사원이 기업 종합 보고서 PDF를 업로드하면, AI가 재무 데이터를 분석하여
**"지금 이 대표에게 어떻게 말을 꺼내야 하는가"** 를 알려주는 툴.

핵심 가치:
- 분석에 드는 시간을 줄인다 (PDF 업로드 → 브리핑 완성까지 1분 이내)
- 영업사원의 전문성을 높여준다 (수치 근거가 있는 멘트 제공)
- 대표에게 건낼 자료를 즉시 만들어준다 (PDF 한 장 출력)

---

## 페이지 구조

```
/                         → 메인 (업로드 화면)
/auth/login               → 로그인
/auth/signup              → 회원가입
/dashboard/[id]           → 분석 결과 브리핑 화면
/dashboard/[id]/report    → 대표 제출용 PDF 미리보기 및 다운로드
/history                  → 분석 이력 목록
```

## 화면별 핵심 설명

### `/` — 메인 업로드 화면
앱의 진입점. 로그인한 영업사원이 PDF를 드래그앤드롭 또는 파일 선택으로 업로드한다.
업로드 즉시 분석이 시작되고, 진행 상태를 단계별로 보여준다.
분석 완료 시 `/dashboard/[id]`로 자동 이동.

지원 포맷 안내(CRETOP, NICE BizLine, 일반 재무제표)를 간결하게 표시.
과거 분석 이력이 있으면 최근 3건을 하단에 노출하여 재접근 편의 제공.

---

### `/dashboard/[id]` — 분석 결과 브리핑 화면
영업사원이 미팅 전 준비하는 화면. 핵심 정보를 한눈에 파악할 수 있도록 구성.

**섹션 구성:**
1. **회사 헤더** — 기업명, 업종, 종업원수, 신용등급, 업계 순위, 분석 기준일
2. **핵심 지표 카드** — 매출, 영업이익률, 당기순이익, 부채비율 (전년 대비 증감 표시)
3. **3개년 재무 추이** — 매출/영업이익/순이익 테이블 + 미니 차트
4. **재무진단** — 성장성/수익성/재무구조/부채상환능력/활동성 등급 (있는 경우)
5. **핵심 이슈** — 위험(빨강)/주의(노랑) 구분, 각 이슈에 수치 근거 포함
6. **영업 추천 멘트** — 4단계 스크립트 (오프닝 / 문제 구체화 / 위기감 조성 / 클로징)
7. **PDF 보고서 생성** 버튼 — `/dashboard/[id]/report`로 이동

---

### `/dashboard/[id]/report` — PDF 보고서 미리보기
대표에게 직접 건넬 수 있는 인쇄용 자료. A4 1~2페이지.

**포함 내용:**
- 회사 기본 정보 + 분석 기준일
- 핵심 재무 지표 요약
- 3개년 추이 (간결한 테이블)
- 발견된 주요 이슈 (최대 3개)
- 컨설팅 제안 포인트 (강점 기반)

브리핑 화면의 영업 멘트는 포함하지 않음 (영업사원 전용 정보).
"다운로드" 버튼으로 PDF 파일 저장 또는 인쇄 가능.

---

### `/history` — 분석 이력
영업사원이 과거에 분석한 기업 목록. 기업명/분석일 기준 검색 및 정렬 가능.
각 항목 클릭 시 해당 `/dashboard/[id]`로 이동.

---

### `/auth/login`, `/auth/signup` — 인증
이메일/패스워드 기반 로그인 및 회원가입.
비인증 상태에서 보호된 페이지 접근 시 `/auth/login`으로 리디렉트.

## 사용자 플로우 (핵심 시나리오)

```
영업사원 첫 방문
    │
    ▼
회원가입 (/auth/signup)
    │
    ▼
메인 화면 (/) — PDF 업로드 안내
    │
    │  PDF 드래그앤드롭
    ▼
업로드 & 분석 진행 중 (로딩: 업로드 → 파싱 → AI 분석)
    │
    ▼
분석 완료 → 자동 이동
    │
    ▼
브리핑 화면 (/dashboard/[id])
    │
    ├── 영업 멘트 확인 & 복사 → 미팅 준비
    │
    └── PDF 보고서 생성 → 다운로드 → 미팅 자리에서 대표에게 전달
```

## 우선순위 및 MVP 범위

### MVP (1차 출시 필수)
- 회원가입 / 로그인
- PDF 업로드 (CRETOP, NICE BizLine 포맷)
- AI 분석 → 브리핑 화면
- 영업 추천 멘트 (4단계)
- 대표 제출용 PDF 보고서 다운로드
- 분석 이력 목록

### Post-MVP (2차)
- 일반 재무제표 포맷 지원 (generic 파서)
- 멘트 커스터마이징 (영업사원이 직접 수정)
- 분석 결과 공유 링크
- 팀 계정 (여러 영업사원이 같은 조직 내 이력 공유)
- 모바일 최적화
