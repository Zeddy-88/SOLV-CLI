# Tech Debt Tracker

개발 중 발생한 기술 부채를 기록하는 문서.  
"일단 넘어가자"고 결정한 순간을 기록해두고, 나중에 반드시 돌아온다.

---

## 기록 방법

```
### [DEBT-001] 제목
- 날짜: YYYY-MM-DD
- 위치: 파일 경로 또는 기능명
- 내용: 무엇을 임시로 처리했는가
- 이유: 왜 지금 제대로 안 했는가
- 해결 방향: 나중에 어떻게 고쳐야 하는가
- 우선순위: 높음 / 보통 / 낮음
```
